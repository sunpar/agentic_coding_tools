# Summarize Session

Summarize the current session's work for future reference.

## Output Format

Keep it under 200 lines. Include:

### What Changed
- Bullet list of changes made
- Files created/modified/deleted

### How to Run/Verify
- Exact commands to run tests
- How to verify the changes work

### What's Next
- Remaining TODOs
- Suggested next steps
- Any blockers or open questions

## Style
- Be concise
- Include exact commands (copy-pasteable)
- No marketing language
